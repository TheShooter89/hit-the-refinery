# Hit The Refinery

a simple game written in Lua and [love2d](https://love2d.org/) framework. The goal is to guide a missile with the mouse to hit and blow up an oil refinery avoiding air defence

dedicated to all ukrainian men and women bravely fighting on the frontline

## Usage

make sure to have [love2d](https://love2d.org/) installed, then run

```bash
love src/
```

alternatively, if on Linux, you can use `make` to run it

```bash
make love
```
